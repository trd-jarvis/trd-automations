# Outbound Email Automation

Use this skill when generated leads need prospecting email prepared from negative AI visibility analysis.

## Purpose
- Build the negative AI analysis for each lead.
- Generate sleek HTML email previews and Gmail-ready payloads.
- Keep the tone direct with a light touch of humor.

## Command
```bash
npm run outreach:email-prepare -- --client <clientId> --limit 25
```

## Output
- HTML files land in `data/reports/`.
- Gmail payload JSON lands in `data/exports/`.
- Team share jobs are queued automatically.
